# Product Vision

## Goal
Build a lightweight, fully customizable, eDEX-inspired UI for Raspberry Pi 4 (4GB), running as a native binary in fullscreen at boot.

## Non-negotiable constraints
- Native ARM64 binary.
- Fast boot and low RAM footprint.
- Modular UI with replaceable regions.
- Plugin system for user-defined panels.
- Smooth animations with tunable quality profile.

## Primary user outcomes
- Start device and land directly in UI.
- Choose what each UI region shows via config.
- Add and remove plugins without touching core code.
- Reserve dashboard region for Technitium DNS query charts.
